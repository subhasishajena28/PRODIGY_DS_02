🚢 Titanic Dataset — Data Cleaning & Exploratory Data Analysis

Data cleaning and exploratory data analysis (EDA) on the Titanic passenger dataset — exploring the relationships between variables and identifying the patterns and trends that shaped survival.

Built for the **Prodigy InfoTech Data Science Internship — Task 2**.
Sample dataset: [`Prodigy-InfoTech/data-science-datasets` — Task 2](https://github.com/Prodigy-InfoTech/data-science-datasets/tree/main/Task%202), originally sourced from the [Kaggle Titanic competition](https://www.kaggle.com/c/titanic/data).

📌 Objective

1. **Clean** the raw dataset — handle missing values, check for duplicates, and engineer useful features.
2. **Explore** the data through univariate, bivariate, and multivariate analysis.
3. **Identify patterns and trends** — in particular, what characteristics were associated with a passenger surviving the disaster.

📊 The Dataset

891 passenger records, 12 original columns:

| Column | Description |
|---|---|
| `PassengerId` | Unique identifier for each passenger |
| `Survived` | Survival (0 = No, 1 = Yes) — **target variable** |
| `Pclass` | Ticket class (1 = 1st, 2 = 2nd, 3 = 3rd) — proxy for socio-economic status |
| `Name` | Passenger name (title embedded, e.g. Mr., Mrs., Miss.) |
| `Sex` | Gender |
| `Age` | Age in years |
| `SibSp` | # of siblings / spouses aboard |
| `Parch` | # of parents / children aboard |
| `Ticket` | Ticket number |
| `Fare` | Passenger fare |
| `Cabin` | Cabin number |
| `Embarked` | Port of embarkation (C = Cherbourg, Q = Queenstown, S = Southampton) |

## 🧹 Data Cleaning

| Column | Missing | Strategy |
|---|---|---|
| `Age` | 177 (19.9%) | Imputed with the **median age within each `Pclass` + `Sex` group** |
| `Cabin` | 687 (77.1%) | Too sparse to impute honestly → engineered `Has_Cabin` (flag) and `Deck` (first letter) instead, then dropped |
| `Embarked` | 2 (0.2%) | Imputed with the mode (`S`, Southampton) |

No duplicate rows or duplicate `PassengerId` values were found.

**Engineered features:** `Title` (extracted from `Name`), `FamilySize` / `IsAlone` (from `SibSp` + `Parch`), `AgeGroup` and `FareBin` (binned versions), `Has_Cabin` and `Deck` (from `Cabin`).

- **Sex was the strongest predictor of survival.** Women survived at **73.9%** vs. **18.9%** for men — a 55-point gap, consistent with the "women and children first" evacuation protocol.
- **Class mattered almost as much**, in a clean staircase: 1st class **63.0%** → 2nd **47.3%** → 3rd **24.0%**.

<img src="images/11_pclass_sex_survival.png" alt="Survival rate by class and sex" width="60%">

- **Class and sex compound each other**: 1st-class women survived at the highest rate of any group; 3rd-class men at the lowest.

- **Children (≤12) survived at 58.0%**, well above the 35.6% rate for young adults (19-35).
- **Family size had a "sweet spot":** travelling alone → 30.2% survival; small families of 2-4 → best at 57.9%; large families of 5+ → worst at 16.1%.
- **A recorded cabin number** (mostly 1st/2nd class passengers) meant a 66.7% survival rate, vs. 29.8% for passengers with no cabin on record.

<img src="images/10_title_survival.png" alt="Survival rate by title" width="60%">

- **Titles extracted from `Name` tell the whole story in one field:** `Mrs` (79.4%) and `Miss` (69.7%) survived best, `Master` — young boys (57.5%) — next, then mixed `Rare` titles (34.8%), with `Mr` surviving worst by far (15.7%).

<img src="images/12_correlation_heatmap.png" alt="Correlation heatmap" width="60%">

- **`Pclass` (-0.34), `Has_Cabin` (+0.32), and `Fare` (+0.26) are the strongest numeric correlates of `Survived`** — again pointing to socio-economic status as a central factor, on top of sex and age.

> 📓 The full notebook contains 14 charts in total (survival overview, demographics, fare/age distributions, embarkation port, and a full pairplot) plus the code and commentary behind every number above.

**Bottom line:** survival on the Titanic was overwhelmingly shaped by *who you were* (sex, age) and *where you could afford to be on the ship* (class, fare, cabin) — a data-backed illustration of how the evacuation protocol interacted with existing social stratification.

🚀 How to Run

```bash
1. Clone the repository
git clone <your-repo-url>
cd titanic-eda-project

2. Install dependencies
pip install -r requirements.txt

3a. Open the notebook (recommended -- includes narrative + all charts)
jupyter notebook notebooks/Titanic_EDA.ipynb

3b. ...or run the equivalent standalone script
cd src
python eda.py
```

Both entry points read `data/titanic.csv`, save the cleaned dataset to `data/titanic_cleaned.csv`, and regenerate every chart in `images/`.
